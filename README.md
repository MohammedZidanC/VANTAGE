# VANTAGE

A modern full-stack task management web app with a sleek map-based UI, smooth transitions, and an integrated admin panel.

---

## 🌐 Live Demo

Frontend (Vercel):  
https://vantage-by-zidan.vercel.app

Backend (API):  
https://mohammedzidanc.pythonanywhere.com

---

## ✨ Features

- 🔐 User Authentication (Login / Register)
- ✅ Task Management (Add, Complete, Delete)
- 🧠 Persistent storage using SQLite
- 🛠 Admin Panel to view all users and tasks
- 🗺 Interactive map-based background UI
- ⚡ Smooth transitions and loading animations
- 🔄 Real-time task updates

---

## 🛠 Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python (Flask)
- SQLAlchemy
- SQLite

### Deployment
- Vercel (Frontend)
- PythonAnywhere (Backend)
